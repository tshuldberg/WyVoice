# MyVoice Landing Page

Basic static marketing page for MyVoice.

## Run Locally

From `/Users/trey/Desktop/Apps/MyVoice`:

```bash
python3 -m http.server 5500
```

Then open:

`http://127.0.0.1:5500/website/`

## Configure Donations

Update these links in `/Users/trey/Desktop/Apps/MyVoice/website/index.html`:

- Buy Me a Coffee link (`https://buymeacoffee.com/TreyTre`)
- Venmo link (`https://venmo.com/u/YOUR_VENMO_HANDLE`)

Recommendation: keep both.

- Primary CTA: Buy Me a Coffee (recurring support + creator profile).
- Secondary CTA: Venmo (fast one-time mobile donations).
